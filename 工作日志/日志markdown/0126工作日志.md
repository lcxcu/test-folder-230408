	2. 减小神经网络大小： 第一个隐藏层有100个节点，第二层有50个，激活函数用identity，权重优化求解器用lbfgs，**惩罚系数为0.1，最大迭代次数为100**。
```python
from sklearn.neural_network import MLPRegressor
fit1 = MLPRegressor(
        hidden_layer_sizes=(100,50), 
        activation='identity',
        solver='lbfgs',alpha=0.1,max_iter=100)
```
